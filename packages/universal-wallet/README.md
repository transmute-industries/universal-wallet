# @transmute/universal-wallet

```
npm i @transmute/universal-wallet --save
```

## Getting Started

```
git clone git@github.com:transmute-industries/verifiable-data.git
npm packages/universal-wallet; // note that this is a mono repo
npm i
npm run lint
npm run test
npm run build
```
